On-page analysis, XML sitemaps, structured data, OpenGraph, AI-powered optimization, redirect manager, local SEO, analytics and more.

### Features

#### ✅ Free Features
- Advanced On-Page SEO Analysis (26-point scoring)
- XML Sitemap Generator + Image Sitemap
- Schema.org Structured Data (Article, FAQ, LocalBusiness, etc.)
- OpenGraph & Social Meta Tags
- Internal Link Suggestions
- Redirect Manager
- Instant Indexing (IndexNow)
- GEO (Generative Engine Optimization) — llms.txt, AI crawler controls
- Breadcrumbs & Speakable Schema

#### ⭐ Pro Features
- AI Meta Generator (Claude AI)
- AI Keyword Research & Content Optimization
- Real Google Search Console keyword data
- Bulk SEO Editor
- Advanced Analytics Dashboard
- Auto-Redirect on slug change
- Local SEO enhancements

### Installation

1. Download the latest release from this repository
2. Upload the `dadsfam-seo` folder to `/wp-content/plugins/`
3. Activate **DadsFam SEO** in WordPress
4. Go to **DadsFam SEO → Dashboard** to get started

### Quick Start

1. Enter your Google Maps / Search Console keys (if needed)
2. Edit any post/page — the SEO meta box appears below the editor
3. Run On-Page Analysis and follow the recommendations
4. Generate and submit your sitemap
