### Installation

1. Download the latest release from this repository
2. Upload the `dadsfam-seo` folder to `/wp-content/plugins/`
3. Activate **DadsFam SEO** in WordPress
4. Go to **SEO → Dashboard** in your admin menu

### Quick Start

1. After activation, visit **SEO → On-Page Analysis**
2. Run a scan on your posts and pages
3. Configure settings under **SEO → Settings**
4. Generate and submit your sitemap to Google Search Console
